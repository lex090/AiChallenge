Навигация по веткам: какой паттерн?

Как пользователь видит и переключает ветки в чате

A

Tab Bar над чатом

Горизонтальные табы с именами веток над областью сообщений. Активный таб подсвечен. Кнопка "+" для создания ветки. Похоже на табы браузера.

Плюсы

  • Привычный паттерн
  • Мгновенное переключение
  • Видно все ветки сразу

Минусы

  • Не показывает иерархию (ветка от ветки)
  • При 5+ ветках — переполнение
B

Branch Indicator + Dropdown

Компактный индикатор текущей ветки в хедере чата (иконка ветки + имя). По клику — dropdown со списком всех веток, их иерархией и действиями (переключить, удалить, создать).

Плюсы

  • Компактно — не занимает место
  • Может показать дерево веток
  • Масштабируется на любое кол-во

Минусы

  • Ветки скрыты за кликом
  • Нужен дополнительный клик для переключения
C

Branch Panel (боковая/нижняя панель)

Отдельная панель (справа или снизу) с деревом веток. Всегда видна или сворачивается. Показывает полную иерархию: main → branch-A → branch-B. Точки ветвления кликабельны.

Плюсы

  • Полная визуализация дерева
  • Удобно для сложных иерархий
  • Можно показать превью ветки

Минусы

  • Занимает экранное пространство
  • Может быть избыточно для 2-3 веток
